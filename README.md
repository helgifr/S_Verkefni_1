# S_Verkefni_1
Big assignment 1

Til að keyra verkefnið skal klóna það í möppu með skipuninni "git clone https://github.com/Ytreg/S_Verkefni_1.git". Síðan skal fara í möppu verkefnisins í terminal og skrifa "npm install" og síðan "npm run dev".

Verkefnið er skipt í nokkrar scss skrár sem er geymt í scss möppu, það var gert til að það væri auðveldara að vinna með.
